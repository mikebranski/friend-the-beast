Installing Additional Mods

Installation of the additional mods is easy, requiring one download and a config tweak.

  1. Download a zip of the mods: https://drive.google.com/file/d/0Bxc6Fv7mZPG8LTA3SnJUZHFKNzQ/view (thanks James!)
  2. Extract the contents to the direwolf20_17\minecraft\mods where you installed FTB -OR- from the launcher you can edit the modpack and add mods and it will move the files there.
  3. Once this is done, start the game (this will create the configs, etc). Note: it will crash this time.
  4. Now, open the ReactorCraft.cfg file located in the direwolf20_17\minecraft\config\Reika directory.
  5. Locate the line I:"Radiation Effect ID"=31 at the end of the file and change the ID from 31 to 82.
  6. Save and close the config file.

That's it! Launch the game again and you're all set. See you topside.

FAQ

Q. Halp! My waypoints disappeared!

A. Rei Minimap stores all of your waypoints locally and they are based on the server host. So, if the host/IP address changes your waypoints will disappear. Fortunately, you can rename the .points files and get them all back!
